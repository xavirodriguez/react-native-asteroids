# React Native Asteroids & Retro Arcade

An arcade engine built with React Native, ECS, and dual-renderer support (Canvas & Skia). Aimed at reducing overhead and supporting reproducible simulations under specific operational constraints.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- npm or yarn
- Expo Go (for mobile testing)

### Installation
```bash
# Install dependencies
pnpm install

# Start the client (Expo)
pnpm start

# Start the server (Colyseus)
cd server
pnpm install
pnpm run dev
```
## 🛠️ Testing

```bash
# Run client tests
pnpm test

# Run server tests
cd server
pnpm test
```
